# Portfolio — Ayoub Seghir

Portfolio personnel bilingue (FR / EN) présentant mon parcours, mes compétences et mes projets.

🔗 **[Voir le portfolio en ligne](https://sahygo.github.io/Portfolio/)** *(si déployé)*

---

## Stack

- HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- Aucune dépendance externe — zéro framework

---

## Fonctionnalités

- **Bilingue FR / EN** — bascule instantanée via le sélecteur de langue
- **Mode sombre / clair** — persisté en localStorage
- **Animation typed** — phrases défilantes adaptées à la langue active
- **Projets séparés** : personnels (SAHYGO, TradeBot) & académiques (SAÉ BUT)
- **Timeline parcours** — expériences et formation
- **Section contact** — liens mail, LinkedIn, GitHub + badges de disponibilité
- **Design responsive** — mobile-first

---

## Lancer en local

Ouvrir `index.html` directement dans un navigateur, ou via un serveur local :

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```
